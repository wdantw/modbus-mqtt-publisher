В диспетчере устройств  версия драйвера должна быть 3.5.2019.1 (представленная здесь)
по умолчанию windows ставит последнюю 3.8.2023.2 она не работает четез .net и в native программах перестает работать если отключитьсяч и подключиться заново
необходимо произвести установку для каждого порта (перевоткнули в другой USB порт - установить заново)
установка:
в диспетчере устройств открыть свойства ком порта, вкладка драйвер
обновить драйвер
найти на этом компьюторе
выбрать с диска (указать inf файл) для первой установки или выбрать из доступных для последующих
